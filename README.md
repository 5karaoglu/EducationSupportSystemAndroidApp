# EducationSupportSystemAndroidApp
Closed system android application for schools.In a few words, it's Instagram for schoolworks.
This app provides you to, see your classes and issues that you are responsible and interact with them. 
Also you can see other people's profile and contact them.

In this app i used these: <br>
Hilt<br>
Flows<br>
Coroutines<br>
MVVM Architecture<br>
Firebase Auth, Storage, Realtime Database<br>
[CircularImageView](https://github.com/lopspower/CircularImageView)<br>
